# Сайт лестниц с админкой

## Что внутри
- `index.html` — сайт
- `admin/` — админка Decap CMS
- `content/site.json` — цены, акции, контакты, преимущества, услуги
- `content/works/` — работы
- `content/posts/` — статьи
- `assets/img/` — изображения
- `assets/js/app.js` — логика сайта и калькулятор

## Важно
Админка по паролю НЕ будет работать через простой Netlify Drop.
Для сохранения изменений нужен такой порядок:

1. Создать GitHub-репозиторий
2. Загрузить туда все файлы сайта
3. Подключить репозиторий к Netlify
4. В Netlify включить Identity
5. В Netlify включить Git Gateway
6. Пригласить себя как пользователя
7. Открыть `/admin/` и войти

## Что можно редактировать в админке
- телефон и ссылки
- цены
- акцию
- услуги и преимущества
- работы с фото
- статьи
- коэффициенты калькулятора

## Что можно доработать потом
- формы заявки
- отправку лидов в Telegram/почту
- отдельные рекламные страницы
- расширенный калькулятор
- каталог ограждений и материалов
