# Stair Configurator MVP

## Реализовано
- Добавлен модуль инженерных формул `stair-formulas.js` с автоподбором количества подъёмов, расчётом 2h+b, угла и длины косоура.
- Добавлен модуль материалов `stair-materials.js` для `metal`, `wood`, `concrete`.
- Вынесена логика UI и расчётов в `stair-configurator.js`.
- Обновлён интерфейс `calculator.html` на 4 шага: конфигурация → параметры → геометрия → материалы и цена.
- Добавлен SQL-файл миграции `supabase/migrations/20260417_stair_configurator_mvp.sql`.

## Поддерживается сейчас
- Полный инженерный расчёт: `straight`.
- Базовый инженерный расчёт: `l_turn` + `left/right` + `landing/winders`.
- Структурная подготовка: `u_turn` (с явным статусом not ready).

## Ограничения MVP
- Для `l_turn` используется упрощённое разбиение проёма по маршам.
- Для `u_turn` отображается понятное сообщение о том, что расчёт будет добавлен следующим этапом.
- Ценообразование реализовано по инженерной цепочке (геометрия → материалы → коэффициенты), но пока без расширенных коммерческих сценариев.
